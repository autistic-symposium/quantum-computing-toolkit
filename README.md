## curated quantum computing and cryptography

<br>

#### 👉🏼 quantum computing technology starts a new exciting era in humanity, combining quantum mechanics, security, artificial intelligence, and computing technologies
#### 👉🏼 once quantum computers become available, some of the main asymmetric algorithms used in cryptography will no longer be safe
#### 👉🏼 the purpose of this repository is to keep a list of high-quality curated resources on the current state of quantum computing hardware and software, and its implications for post-quantum cryptography
